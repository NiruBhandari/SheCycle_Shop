SheCycle is a modern web-based hostel thrift and rental store that promotes affordable and sustainable fashion for girls. It enables peer-to-peer exchange of clothing items, helping reduce textile waste while connecting students within their local community.

# Key Features

# Seller Side
. Seller registration with secure login and email verification
. Upload clothing items with multiple images, description, category, price, and rental period
. Manage and update product listings

# Buyer Side
. Browse and request rental/thrift items
. Buyer login & profile management with photo upload
. Location input for connecting nearby users


# Tech Stack Used
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Server: XAMPP Apache
